# CONTEXT.md —— 领域语言

本文件只记录领域术语，不写实现细节（补充计划第 10 节）。实现见 `app/`。

- **创作参考**：系统所有输出的定位。生成内容仅供团队内部创作参考，发布前必须人工确认，不承诺可直接发布。
- **采集任务（Task）**：一次从输入到产出的处理单元。输入为抖音公开视频链接或本地上传的音视频。
- **公开视频**：当前部署环境可访问、且配置 Cookie 后可解析的单条抖音公开视频。不含私密/已删除/无权限/被风控/主页合集批量。
- **转写稿（原始）**：faster-whisper 对音频的原始语音转写输出。
- **人工修订稿**：用户在详情页对转写稿的人工修订版本。下游清洗、结构拆解、改写优先使用它。
- **清洗稿**：对转写稿做去口水词、加标点、分段后的可读文本（两步 LLM 第一步）。
- **结构拆解**：简版结构分析，含开头钩子、核心卖点、节奏转折、结尾行动。
- **改写版本**：基于清洗稿/修订稿生成的三类文案——口播稿、种草笔记文案、标题和开头钩子。
- **导出稿**：把处理结果导出的 Markdown / Docx 文件。
- **人工反馈**：对每条改写结果的“好用 / 不好用 + 原因标签”评价，是质量指标的唯一数据来源。
